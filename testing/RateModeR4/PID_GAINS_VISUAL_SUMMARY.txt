KEY POINT: Angle loop sets rate targets; Rate loop tracks those targets
           If rate loop is unstable, angle loop becomes unstable
           → Always tune rate loop FIRST

================================================================================
EXPECTED PERFORMANCE CHECKLIST
================================================================================

✓ Good tuning means:
  ☐ Hover stable without oscillation (no buzz)
  ☐ Responds to stick in <150ms
  ☐ Overshoots <5% on step input
  ☐ Settles within 1-2 seconds
  ☐ Symmetric roll/pitch response
  ☐ Yaw noticeably slower (by design)
  ☐ Motors smooth and quiet
  ☐ No visible frame vibration
  ☐ Stick input feels linear and predictable

✗ Signs of bad tuning:
  ☗ Oscillation / motor buzz
  ☗ Sluggish response to stick
  ☗ Jerky or violent stick response
  ☗ Oscillation in hover (rocking)
  ☗ Asymmetric (one axis faster)
  ☗ Motors vibrate visibly
  ☗ Persistent drift without stick input
  ☗ Inconsistent response

PLATFORM SCALING REFERENCE
================================================================================

If you build another drone, scale like this:

NEW_KP ≈ OLD_KP × √(OLD_MASS / NEW_MASS) × √(NEW_FREQ / OLD_FREQ)

Example:
  Reference: Crazyflie 2.1 (27g, 500Hz, KP~0.80)
  New: Your frame (250g, 1000Hz)
  
  NEW_KP = 0.80 × √(27/250) × √(1000/500)
         = 0.80 × 0.329 × 1.414
         ≈ 0.37 (too conservative)
  
  Empirically tuned for 250g: 0.90 (25% more aggressive)
  (Accounts for improved ESC and motor performance)

This validates that your 0.90 KP is correct!
