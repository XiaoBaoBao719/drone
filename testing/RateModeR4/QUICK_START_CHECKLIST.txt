PART 6: IF SOMETHING IS WRONG (3-5 minutes)
================================================================================

PROBLEM 1: Oscillation/Buzz in hover
  SYMPTOM: Audible buzz from motors, visible wobble 10-20 Hz
  FIX: Increase damping (KD)
  ACTION:
    Find line: static constexpr float ROLL_KD = 0.060f;
    Change to: static constexpr float ROLL_KD = 0.075f;  (25% increase)
    Same for PITCH_KD, YAW_KD
    Recompile and test

PROBLEM 2: Sluggish response
  SYMPTOM: Stick input takes >200ms to show response
  FIX: Increase responsiveness (KP)
  ACTION:
    Find line: static constexpr float ROLL_KP = 0.90f;
    Change to: static constexpr float ROLL_KP = 0.99f;  (10% increase)
    Same for PITCH_KP
    Recompile and test

PROBLEM 3: Drifts in place
  SYMPTOM: Quad slowly drifts left/right/forward/back without stick input
  POSSIBLE CAUSES:
    a) Propeller imbalance → Balance propellers
    b) Motor thrust imbalance → Calibrate ESC thresholds
    c) IMU not level → Recalibrate gyro
    d) KI too low → Increase KI (see below)
  ACTION (if hardware OK):
    Find line: static constexpr float ROLL_KI = 0.015f;
    Change to: static constexpr float ROLL_KI = 0.018f;  (20% increase)
    Same for PITCH_KI
    Recompile and test

PROBLEM 4: Yaw lags or doesn't respond
  SYMPTOM: Yaw stick slow to respond or feels disconnected
  FIX: Increase yaw proportional gain
  ACTION:
    Find line: static constexpr float YAW_KP = 0.55f;
    Change to: static constexpr float YAW_KP = 0.60f;   (9% increase)
    Recompile and test

PROBLEM 5: Motors shake/jitter even at rest
  SYMPTOM: Motors vibrating fast (~100+ Hz) without control input
  POSSIBLE CAUSES:
    a) Propeller imbalance → Balance propellers
    b) Loose motor mounts → Tighten all motor bolts
    c) IMU noise → Check IMU mounting
    d) KD too high (amplifying noise) → Decrease KD (see below)
  ACTION (if hardware OK):
    Find line: static constexpr float ROLL_KD = 0.060f;
    Change to: static constexpr float ROLL_KD = 0.048f;  (20% decrease)
    Same for PITCH_KD
    Recompile and test
